# GIF File Required

Please place the Rafael Nadal tennis GIF file in this `public` folder and name it `nadal-tennis.gif`.

The GIF should be from:
https://makeagif.com/gif/rafael-nadal-ultimate-slow-motion-compilation-forehand-backhand-serve-2013-cincinnati-open-33Y2Gx

To download:
1. Visit the URL above
2. Right-click on the GIF and "Save image as..."
3. Save it as `nadal-tennis.gif` in this `public` folder

The background will automatically use this file once it's placed here.
